feat(search): add trending repositories endpoint and fix URI encoding

- Implemented `/api/github/search/trending-repositories` in `GithubSearchController`.
- Added `getTrendingRepositories(language, since, page, size)` to `GithubSearchService` to fetch repositories dynamically based on creation date (daily, weekly, monthly) sorted by stars.
- Fixed a bug where `RestTemplate` was double-encoding the query string by changing URL representations from raw `String` to properly built `java.net.URI` instances. This prevents `UriTemplateHandler` from improperly parsing search strings containing special characters like `>` and spaces.
