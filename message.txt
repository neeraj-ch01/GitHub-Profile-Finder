feat: rich analytics & visualization

AnalyticsService.java
A new Spring @Service that fetches all public repositories for a given username.
It calculates an aggregated frequency map for languages (e.g., {"Java": 15, "JavaScript": 5}).
It also identifies the top repositories by Stars and top repositories by Forks.

AnalyticsController.java
Expose a new endpoint: GET /api/github/analytics/{userName}.
This endpoint returns a combined DTO with language distribution, top starred repos, and top forked repos so the frontend charts can consume it instantly.
